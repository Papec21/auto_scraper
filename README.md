# auto_applier

1. GOAL
The goal of this program is to automatically apply on job offers on pracuj.pl every 24 hours.
It will save me some time that I spend on scrolling through offers and applying myself.
When an offer happens to have external link to recruitment, the program will save the link and make a report, so I can apply on the offers myself.
At least for now I'm not planning on making this auto_applier selective in job offers. It won't filter them, just go to desired page with preselected options and scrap every offer. I may add feature like this in the future.

2. WHAT WILL I LEARN?
I'll learn how to scrape data from dynamic websites like pracuj.pl, how to avoid bot detection (playwright_stealth), parse and store the data, automate simple processes and hopefully I'll also learn how to write a solid README.

3. TECHNOLOGIES
Python libraries: asyncio, playwright.async_api, playwright_stealth, psycopg2, csv (It's work in progress)
